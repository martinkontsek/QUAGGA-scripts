# How to build Quagga from Eclipse without optimalization
# Eclipse Neon 4.6.1
#

Project -> Properties -> Autotools -> Configure Settings -> configure -> Advanced -> Compiler Flags -> tick  Debug (-g) -> Apply -> OK
