# How to import Quagga project to Eclipse
#

1. Open Eclipse
2. File -> Import... -> C/C++ -> Existing code as Autotools project -> Next
3. Existing code location -> Browse... -> Browse for Quagga-EIGRP folder
4. Select project language -> C Project -> Next
5. Finish
